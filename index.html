Resume Builder

The Resume Builder is a dynamic web application built using HTML, CSS, and JavaScript that enables users to create, preview, and download professional resumes interactively. It features real-time updates, allowing all entered information to instantly reflect in the live preview. Users can add or remove multiple entries in sections such as education, experience, and projects with ease. The tool also includes high-quality PDF generation using html2canvas and jsPDF, enabling seamless downloads. With its clean UI, efficient validation, and error handling, it ensures a smooth and user-friendly experience.

Employee Retention Analysis

The Employee Retention Analysis Project is a machine learning-based study that predicts and analyzes employee attrition trends using Random Forest and Support Vector Machine (SVM) models. It involves cleaning, preprocessing, encoding, and scaling the dataset for optimal model performance. Multiple SVM kernels (linear, polynomial, and RBF) are compared, with results visualized through accuracy charts, confusion matrices, and feature importance graphs. The project identifies the key factors influencing retention and provides data-driven insights and recommendations, with all findings neatly saved and organized for clear presentation.
